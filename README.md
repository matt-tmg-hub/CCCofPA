# Cornerstone Cattle Company of PA

Website for Cornerstone Cattle Company of PA — a small, family-owned Black Angus farm in Bucks County, Pennsylvania.

## Pages

- `index.html` — homepage
- `store.html` — share reservations and checkout
- `contact.html` — contact form
- `images/` — site photography

## Local preview

Double-click `index.html` to open in a browser. No build step needed — it's plain HTML, CSS, and JavaScript.

## Deployment

Deployed via Vercel. Every push to `main` triggers an automatic deploy.

## Editing content

Open the `.html` files in any text editor (VS Code, Notepad++, even TextEdit) to change copy, pricing, or cut lists.
